## Section 3: Operational Constraints

### 3.1 Temperature-Controlled Cargo (TEMP)

Temperature-controlled cargo requires special handling:
- Maximum TEMP cargo per flight: 15% of total payload
- Pre-cooling must begin 4 hours before loading
- Active containers require power allocation (max 8 per flight)
- Documentation: UN3373 or appropriate DG declaration required

### 3.2 Dangerous Goods

Dangerous goods limitations by aircraft type:
- 747-400F: Class 1-9 (with restrictions)
- 777F: Class 1-9 (with restrictions)
- 767-300F: Class 2-9 only
- A330-200F: Class 2-9 only

### 3.3 Weight and Balance

All flights must comply with:
- Center of Gravity (CG) envelope per aircraft
- Floor loading limits (per position)
- Running load limits for heavy/dense cargo
- Tie-down requirements for heavy single pieces
